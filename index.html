I have some ideas for future projects that I might take on one day. They range from quick easy program to entire videogames.

  • Hackerman
    Wanna look cool in front of everyone without having to resolve to websites like hackertyper and doing it all from the comfort and speed of your terminal? Hackerman wants to do all this. I imagine a system that lets you define a file to show the text from and properties like if you have to type to advance the text or if it scrolls automatically, the appearance and more through arguments, and then run it through ncurses.

  • LIFELINE - A week is a lifetime
    A macabre diary application, displaying how much of your life you have already used up, how long you still have left to live, and a chart showing every week in your life you already lived, inspired by kurzgesagt.

  • A vocabulary trainer
    This would really come in handy sometime. The tool would be a terminal application, again in ncurses. The data should be saved using the ankidroid format or another existing flash cards file format.

  • A Voxel game in which you build a society from scratch
    The Idea is very rough, but it is supposed to be a city builder/simulation which takes on progressively larger scale. You start in a dead universe, as a 3d character, on a large floating rock once called earth. You are an immortal being with otherwise human powers that lives since the 21st Century, now finally pulling itself up to rebuild the universe. You start by building small machines enabling you to do basic things like terraforming a few voxels at a time, creating energy and food. Then you clone your first human and create a society. You can then switch into a top-down view of your planet, terraform at larger scale and manage multiple buildings at once. Then you develop spacefaring, extending to other planets, eventually conquering entire galaxies. You complete the game by building an artifact that can create another universe, and create a new savefile.
    Pretty ambitious, I know. I would want to create an own engine for this game to enable a high-performance simulation of possibly complex societies and machinery, all set in a voxel world where you can modify every single one of the infinite amount of voxels in an infinite universe.

  • A Taskserver
    I am using Taskwarrior, a really cool and featureful CLI to-do list. They also have this project called Taskserver which lets you, as the name suggests, set up a tool on a server that synchronizes all your tasks across all your devices. I currently do this through OneDrive by just having my task folder in my OneDrive directory that I sync regularly with the amazing OneDrive CLI tool by abraunegg, but having a synchronization via my very own server sounds way cooler.
  • A camera system with face detection
    I've always wanted to play around with OpenCV. I'd simply connect a webcam to the pi and let a program run that collects the webcam data, saves an image every 3 seconds or so and tries to get interesting data out of it (how bright is it, are there faces/movement on the image, etc). I don't know if it's a smart idea to pull that off with C++ as well, maybe I'd have to use Python for that, but I like the idea to try and decode the camera data through C++ by myself.
